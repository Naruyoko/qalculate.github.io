Qalculate! downloads
(!) Note that even though most planned features are in place and I personally find it highly stable and reliable in everyday use, Qalculate! definitely needs more testing and no guarantees whatsoever are given. With that said, you are welcome to download the latest version of Qalculate! and report any bugs you find.
Windows binaries:
  • 64-bit installer
  • 64-bit "portable" zip package
  • 32-bit installer
  • 32-bit "portable" zip package
Includes the graphical user interface and CLI, and all dependencies, except Gnuplot. The new Qt UI requires requires Windows 10, while the classic GTK GUI requires Windows 7, or higher.

Use compatibility mode to avoid keyboard input issues with 32-bit version.

The portable file can be unpacked and run without further installation. No application data are read or written outside of the unpacked directory.

The 64-bit binaries can also be installed using the Windows Package Manager (winget) using the command "winget install qalculate".

Linux binaries:
A snap packages, with all dependencies included, is available at snapcraft.io/qalculate (GTK GUI) and snapcraft.io/qalculate-qt (Qt GUI).
Use the above link or install with the command "sudo snap install qalculate" or "sudo snap install qalculate-qt".

A flatpak package (GTK GUI) is available at Flathub (flathub.org/apps/details/io.github.Qalculate).
Click the above link or use the command "flatpak install https://flathub.org/repo/appstream/io.github.Qalculate.flatpakref".

Self-contained binaries (GTK GUI): qalculate-4.0.0-x86_64.tar.xz
Extract and run from anywhere on most GNU/Linux distributions. Does not include any languages other than English.

Most distributions for Linux provide packages (which may or may not be outdated) for Qalculate!, and all its dependencies, which you can install using the distribution's standard installers.

Mac OS binaries:
  • Qalculate! at MacPorts
  • Qalculate! Homebrew Formulae
Source code:
  • Library and text interface (libqalculate)
  • Graphical user interface (GTK)
  • Graphical user interface (Qt)
Library requirements:
  • GMP, MPFR, Libxml2 (required)
  • libcurl, icu, gettext (strongly recommended)
  • Gnuplot (optional, for plotting)
Graphical user interface requirements:
  • GTK (>= 3.10) or Qt (>= 5.6)
  • libqalculate (>= 4.0.0)
Text interface requirements:
  • GNU Readline (recommended)
  • libiconv (recommended)

Old versions can be found at the project page for libqalculate and qalculate-gtk. Ancient versions are available at the old project page at sourceforge.net.
