Qalculate! downloads


(!) Note that even though most planned features are in place and I personally find it highly stable and reliable in everyday use, Qalculate! definitely needs more testing and no guarantees whatsoever are given.

With that said, you are welcome to download the latest version of Qalculate! and report any bugs you find.

Qalculate! version 3.11.0 (source code):

  • Library and text interface (libqalculate)
  • Graphical user interface (qalculate-gtk)

Library requirements:

  • GMP, MPFR, Libxml2 (required)
  • libcurl, icu, gettext (strongly recommended)
  • Gnuplot (optional, for plotting)

Graphical user interface requirements:

  • GTK+ (>= 3.10)
  • libqalculate (>= 3.8.0)

Text interface requirements:

  • GNU Readline (recommended)
  • libiconv (recommended)

Windows binaries:

  • 64-bit installer
  • 32-bit installer
  • 32-bit "portable" zip package
Includes the graphical user interface and CLI, and all dependencies, except Gnuplot. Requires at least Windows Vista or higher (has been tested on Windows 7 and Windows 10).

Use compatibility mode to avoid keyboard input issues with 32-bit version.

The portable file can be unpacked and run without further installation. No application data are read or written outside of the unpacked directory.

The 64-bit binaries can also be installed using the Windows Package Manager (winget).

Linux binaries:

A binary snap package, with all dependencies included, is available at snapcraft.io/qalculate for those using a slow moving GNU/Linux distribution (install with the command "sudo snap install qalculate").

A binary flatpak package is also available (github.com/scx/qalculate-flatpak).
Single-file bundle: qalculate-gtk-3.11.0-x86_64.flatpak (installation command: flatpak --user install "qalculate-gtk-3.11.0-x86_64.flatpak")

Self-contained binaries: qalculate-3.11.0-x86_64.tar.xz (NEW)
Extract and run from anywhere on most GNU/Linux distributions.

Most distributions for Linux provides packages (which may or may not be outdated) for Qalculate!, and all its dependencies, which you can install using the distribution's standard installers.


A Mac OS X port is available here:
Qalculate! at MacPorts.


Old versions can be found at the project page for libqalculate and qalculate-gtk. Ancient versions are available at the old project page at sourceforge.net.
